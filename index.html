My Involvement in Outhire

ScreeningQueston is a component where talent teams (users) can edit screening questions that will be filled out by candidates.There are 5 types of questions: Short (Single line) questions, Long (Paragraph) questions, Multiple Checkbox questions, Single Checkbox questions and Toggle questions. Each question can be marked as mandatory and can be moved up or down. When users edit the screening questions, these questions will be displayed on the application form that can be accessed by candidates. Mandatory questions that remain unanswered will be highlighted with a tooptip for users when they submit the form. The answers to the screening questions will be saved to databased can be rendered on a route called Candidate where users can view the answers and screen candidates. You can play with the ScreeningQuestion component at here.

On “Job Post” page there is a text editor that can detect and highlight gender-coded words and words that promote diversity and emphasise a positive work environment . To accurately detect the nuances of each word based on context, I trained a NLP model spaCy to make the text editor identify word usage in a more precise manner. I also did Depth First Search to ensure that the highlights in the text editor move seamlessly regardless of the format and the number of bullet points in any job description. This means that if a user adds or deletes any part of the text and this affects the position of highlighted words, the highlights should adjust to the new index of the highlighted words. You can play with the text editor by signing up at Outhire and creating a free AI-generated job description.
